# **App Name**: Simple Site

## Core Features:

- Content Display: Display provided text and media content in a structured format.
- Contact Form: Implement a basic form for users to submit inquiries.

## Style Guidelines:

- Primary color: Calm blue (#75A9D9) to convey trustworthiness.
- Background color: Very light blue (#EBF4FA), a desaturated version of the primary color, to ensure a clean and professional backdrop.
- Accent color: Muted violet (#9975D9) to add visual interest without overwhelming the user.
- Body and headline font: 'Inter', a sans-serif typeface that offers excellent readability.
- Simple, outlined icons to complement the minimalist design.
- Clean and well-spaced layout to improve readability and user experience.
- Subtle transitions and hover effects to provide visual feedback.
